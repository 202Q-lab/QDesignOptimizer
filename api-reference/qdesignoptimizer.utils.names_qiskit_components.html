qdesignoptimizer.utils.names_qiskit_components.QiskitComponentName

A string representing the name of a Qiskit Metal component, by convention starts with ‘name_’

Examples

>>> "name_qubit"
>>> "name_resonator_1"
>>> "name_coupler_1to2"
